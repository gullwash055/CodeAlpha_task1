# Task 2 — Sensor-Based IoT Simulation: Light-Activated LED

## CodeAlpha IoT Internship

## Overview
This project simulates a light-activated LED system using a Tinkercad 
circuit built around an Arduino UNO. A photoresistor (LDR) continuously 
measures the surrounding light level, and the Arduino automatically 
switches an LED on when the light level drops below a defined threshold, 
and switches it off again once enough light is detected — mimicking the 
working principle of a real-world automatic streetlight or dark-sensing 
security light.

## Components Used
- Arduino UNO
- Photoresistor (LDR)
- LED
- Resistors
- Breadboard and jumper wires

## Key Topics Covered
- Circuit design and connections
- Arduino programming logic
- Real-world application: automatic streetlights / security lighting
